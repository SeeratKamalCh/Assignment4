Program to remove green background from an image.

The images folder contains the images with green background 

The removed folder contains the images with removed green background.


Steps of program execution:

When the remove_background.py file runs, the program shows images present in the images folder.

Type the name of any image in the "images" folder for example "car.jpg"

The output image with removed green background will be saved in the "removed" folder with filename "removed_" + 
original image filename for example for car.jpg it will be "removed_car.jpg"
